## 🧭 Что это

SELFVPS — доработанный десктоп-клиент AmneziaVPN для своего VPS. Базовый Amnezia отличный, но на IP-адресах, которые РКН уже пометил, часть протоколов душится: reality/TLS клампится на любом порту, голый UDP дропается. Здесь допилены ядро и маршрутизация так, чтобы туннель держался **даже на флагнутом IP**, а раздельное туннелирование реально работало и по приложениям, и по сайтам.

## ⭐ Чем отличается от оригинала

### 🔓 XRay, который переживает РКН на флагнутом IP
- VLESS поверх **mKCP + FinalMask salamander** — та же обфускация, что держит Hysteria: каждый пакет превращается в псевдослучайный шум, DPI/ТСПУ не может его зафингерпринтить (в отличие от reality, чей TLS-хендшейк на флагнутом IP душат на **любом** порту).
- Работает на **высоком UDP-порту** (443/QUIC душат жёстко — высокий порт проходит).
- Reality оставлен в коде для чистого IP / CDN-фронтинга.

### 🧩 Раздельное туннелирование, которое работает
<img src="docs/screenshots/connection.png" width="430" align="right" alt="Раздельное туннелирование"/>

- **По приложениям** — починен WFP-драйвер (раньше не запускался без killswitch, и app-split молча не работал).
- **По сайтам** — доменные маски, `geoip:ru → напрямую`, авто-обход РКН-листа; всё in-core в XRay, реактивный UI (список больше не «серый»).
- **KillSwitch не рубит локалку** — LAN остаётся доступной при поднятом VPN.

<br clear="right"/>

### 🔁 Мультидевайс без боли
Общий персистентный UUID + salamander-пароль в host-volume: переустановка контейнера **не ломает другие устройства**.

### ⚡ Тюнинг под нестабильный трансграничный канал
- Hysteria2 на **BBR** (адаптивный) вместо Brutal — фиксированная скорость коллапсит на переменном RKN-канале.
- mKCP на стоковых параметрах (`capacity` — жёсткий MB/s-потолок, занижать нельзя).

### 🎨 Редизайн интерфейса
Тёмная палитра в духе GitHub + синий акцент, шрифт Inter, щит-индикатор соединения, левый сайдбар на десктопе.

